# 🐕 Dogs and facts
[![API](https://img.shields.io/badge/API-21+-brightgreen.svg)](https://android-arsenal.com/api?level=21)

Мобильное приложение для Android, предоставляющее информацию о породах собак и интересные факты о них. Разработано с использованием современных подходов к мобильной разработке: **Jetpack Compose**, **MVVM**, **Coroutines** и **Retrofit**.

## 📱 Возможности приложения

### 📋 Каталог пород
- Просмотр списка пород собак
- **Поиск в реальном времени** с фильтрацией по названию
- **Раскрывающиеся карточки** с детальной информацией:
  - Описание породы
  - Продолжительность жизни
  - Вес (для самцов и самок)
  - Информация о гипоаллергенности

### 🎲 Случайная порода
- Генерация случайной породы по нажатию кнопки
- Отображение полной информации о выбранной породе
- Красивый UI с анимациями и иконками

### 💡 Случайный факт
- Получение интересных фактов о собаках
- Простой и интуитивный интерфейс

### 🧭 Навигация
- Нижняя панель навигации (Bottom Navigation Bar)
- Плавные переходы между экранами
- Сохранение состояния экранов

## 🛠️ Технологический стек

### Основные технологии
- **Язык:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Архитектура:** MVVM (Model-View-ViewModel)
- **Навигация:** Navigation Component

### Библиотеки и зависимости
- **Retrofit 2** — типобезопасный HTTP-клиент для работы с REST API
- **OkHttp** — логирование сетевых запросов
- **Gson** — парсинг JSON
- **Kotlin Coroutines & Flow** — асинхронное программирование
- **StateFlow** — реактивное управление состоянием
- **ViewModel** — хранение и управление UI-данными
- **Material Design 3** — современный UI-дизайн
- **Material Icons Extended** — расширенный набор иконок
